# Copy GIF - Chrome Extension

A Chrome extension that allows you to Copy GIFs to your clipboard with their animation intact.

## Features

- Right-click context menu option to Copy GIFs
- Preserves GIF animation when copying (unlike native "Copy Image")
- Works on any website with GIF images
